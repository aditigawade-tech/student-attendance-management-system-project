🔹**Problem Statement**

Managing student data manually can lead to errors and inefficiency. There is a need for a simple program to store student details and calculate marks accurately.

🔹**Key Features**

Input student details (Name, Roll Number, Marks)
Calculate total and average marks
Display student record in a structured format
Simple and easy-to-use command-line interaction

🔹 **Objectives**

To automate student record management
To reduce calculation errors
To improve data organization
To make the system simple and user-friendly

🔹 **Technologies Used**
Python (Core Programming Language)
Command Line Interface (CLI)
VS Code (Development Environment)
